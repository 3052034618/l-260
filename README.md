# 企业资产折旧测算工具

一个供财务人员在编制预算时快速比较不同购置方案年度影响的桌面插件。

## 功能特性

### 四个核心窗口
- **资产导入**：支持粘贴导入、Excel导入、手动新增，可批量调整参数
- **参数设置**：选择折旧方式、设置残值率、启用日期、预算范围等
- **测算结果**：月度折旧明细、部门费用汇总、即将折旧完毕资产标记
- **方案对比**：多采购方案对比、可视化图表分析

### 核心功能
- ✅ 三种折旧方式：直线法、双倍余额递减法、年数总和法
- ✅ 资产清单粘贴导入（支持Tab/逗号分隔）
- ✅ Excel文件导入导出
- ✅ 启用日期设置
- ✅ 残值率设置（支持批量调整）
- ✅ 使用年限批量调整
- ✅ 月度折旧精确计算
- ✅ 部门费用汇总
- ✅ 多采购方案对比
- ✅ 即将折旧完毕资产标记
- ✅ 测算明细导出
- ✅ 预算摘要生成
- ✅ 常用参数保存/加载
- ✅ 本地数据存储，不影响正式账务

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **桌面框架**：Electron
- **Excel处理**：SheetJS (xlsx)

## 使用说明

1. **资产导入**
   - 点击"下载模板"获取标准模板
   - 填写模板后点击"导入Excel"上传
   - 或直接从Excel复制数据到粘贴框，点击"解析并导入"
   - 也可点击"新增资产"手动添加

2. **参数设置**
   - 选择默认折旧方式
   - 设置使用年限和残值率
   - 配置预算开始日期和年度数
   - 可将参数保存为常用配置

3. **测算结果**
   - 系统自动计算，点击"重新测算"刷新
   - 查看资产明细、部门汇总、类别汇总
   - 查看月度折旧趋势图表
   - 导出测算明细和预算摘要

4. **方案对比**
   - 保存当前配置为方案
   - 选择多个方案进行对比
   - 查看首年折旧、总额、年度趋势对比
   - 导出对比报告

## 折旧方式说明

### 直线法 (平均年限法)
年折旧额 = (原值 - 残值) / 使用年限
特点：每年折旧额相等，适用于使用均衡的资产。

### 双倍余额递减法
年折旧率 = 2 / 使用年限 × 100%
特点：前期折旧多，后期少，适用于技术更新快的资产。

### 年数总和法
年折旧率 = 尚可使用年限 / 年数总和 × 100%
特点：前期折旧多，后期少，适用于更新换代快的资产。

## 数据安全

- 所有数据保存在本地浏览器存储中
- 无需联网，不涉及外部服务器
- 可随时导出备份或保存方案
- 不会影响正式财务系统数据
